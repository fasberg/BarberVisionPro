# ✂️ BarberVision PRO

**BarberVision PRO** — це інтелектуальна система для автоматизації роботи сучасних барбершопів. Застосунок використовує технології комп’ютерного зору (Computer Vision) та нейронні мережі для точного антропометричного аналізу обличчя клієнта в режимі реального часу.

Проєкт розроблений для допомоги майстрам-барберам в об'єктивному виборі стилю зачіски на основі математичних параметрів та штучного інтелекту.

## 🌟 Основні можливості
* **AI-аналіз обличчя:** Детекція 468 ключових точок за допомогою моделі MediaPipe Face Mesh.
* **Математична точність:** Розрахунок морфологічного коефіцієнта (Ratio) для точної класифікації форми обличчя (Oval, Square, Round, Heart, Oblong).
* **Система рекомендацій:** Автоматичний підбір стилів зачісок, що інтегрований з базою даних.
* **Генерація звітів:** Створення персоналізованих PDF-документів із результатами аналізу для клієнтів.
* **Цифрова історія:** Ведення архіву відвідувань та можливість експорту даних у формат CSV.
* **Багатомовність:** Повна локалізація інтерфейсу (Українська / Англійська).

## 🛠 Технологічний стек
* **Мова:** Python 3.9+
* **AI/CV:** MediaPipe, OpenCV
* **GUI:** Tkinter (Custom Dark Theme)
* **База даних:** SQLite3
* **Звітність:** FPDF, Pillow

## 🎮 Керування системою
Програма реагує на гарячі клавіші під час роботи вікна камери:

| Клавіша | Функція |
| :---: | :--- |
| **S** | Зберегти миттєвий скріншот аналізу |
| **P** | Сформувати PDF-звіт та занести запис в історію БД |
| **C** | Експортувати повну історію клієнтів у файл CSV |
| **Q** | Закрити модуль аналізу та повернутися в меню |

## 📦 Встановлення та запуск
1. Клонуйте репозиторій на свій ПК.
2. Встановіть залежності: `pip install -r requirements.txt`.
3. Запустіть головне меню: `python main_menu.py`.
